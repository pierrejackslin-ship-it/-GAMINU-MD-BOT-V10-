## ✨ Main Features

### 🎧 Media / Download
- `.play <music>` → Download YouTube as MP3  
- `.tiktok <link>` → Download TikTok videos  
- `.lyrics <music>` → Get song lyrics  

### 🌍 Translation
- `.trad <lang> <text>` → Translate text to any language  

### 🛡️ Security / Protection
- `.antidelete on/off` → Prevent deleted messages  
- `.antispam on/off` → Anti-spam system  
- `.antivirus on/off` → Basic protection  

### 👁️ View Once / Decode
- `.vv3` → Decode view-once messages  
- 🔐 Works only on the bot number  

### 📊 Status Features
- 👁️ Auto view status  
- ❤️ Auto like status  
- 📥 Save statut 
- 📥 Full status save
